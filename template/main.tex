% possible values are:
%   * 169  => 16/9  = 1.77(7)  [wide-screen panoramic TV, eg. 1920x1080]
%   * 1610 => 16/10 = 1.60     [typical panoramic monitor, eg. 1920x1200]
%   * 149  => 14/9  = 1.55(5)
%   * 32   =>  3/2  = 1.50
%   * 43   =>  4/3  = 1.33(3)  [typical non-panoramic monitor, eg. 1600x1200]
%   * 54   =>  5/4  = 1.25
%
